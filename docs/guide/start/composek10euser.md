    在使用者的public/env.js配置URL，配置文件如下
    ```
    window.env = {
        CoRepoUrl: '',
        HasuraUrl: '',
        ApiUrl: '',
        OIDC_AUTHORITY: '',
        OIDC_CLIENTID: '',
        OIDC_CLIENTSECRET: '',
    };
    ```
