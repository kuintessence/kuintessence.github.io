<p align="center"><font size=5>坤仪·万象-内容库</font></p>

:  进一步扩展内容仓库丰富度，坤仪·万象还建立了一个开放式内容仓库平台，用户无需登录坤仪·万象即可在该平台存储计算编排所需的内容例如 软件描述、软件用例描述、工作流模板等。

:  用户可自行上传、分享和管理自己的软件或算例信息。
内容仓库提供可视化内容编辑器，用户可轻松地将自己的知识转换成标准的描述。
